to.int

Cast any expression to integer
Uses php (int)expr


Parameters

u0 or expr: expression


Returns

The Integer Value


Errors

No expression Provided.
PHP might throw error if it cannot be casted to int
[code.run] output: [to.int 100 /] [/code.run] [code.run] [to.int] 200 [/to.int] [/code.run] 
is.int

Check whether expr is integer or not.
If content is provided and the expr is an integer the content will be parsed
Uses php (int)expr


Parameters

u0 or expr: expression
content : The content to parse if the expr is integer

Returns

True or False


Errors

No expression Provided.
PHP might throw error if it cannot be casted to int
[code.run] output: [is.int i:100 /] [/code.run] [code.run] [is.int i:200] I am an int [to.int 40 /] [/is.int] [/code.run] 
int.plus

plus : The integer to add

Returns

Errors
